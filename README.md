# number-guessing-game
Number Guessing Game:
A console-based number guessing game with three difficulty levels. The program picks a random number and you have to guess it, with hints telling you to go higher or lower until you get it right!

Live Demo:

Play here: https://onlinegdb.com/HyynBT_Zg

Features:

Three difficulty levels: Easy (1–10), Medium (1–100), and Hard (1–500)
Hints after every wrong guess (try higher / try lower)
Input validation (no invalid inputs accepted)
Play again option after winning


How to Run Locally:
Requirements:

Python 3.x installed on your machine

Steps:

Clone or download this repository
Open a terminal in the project folder
Run the following command:

python "Number Guessing Game.py"

How to Play:

When prompted, type easy, medium, or hard to select a difficulty
Enter your guess when prompted
The game will tell you if you need to go higher or lower
Keep guessing until you find the correct number!
When you win, choose to play again or exit
